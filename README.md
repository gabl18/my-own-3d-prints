# my-own-3d-prints
Learning some Blender and PicoCad to make my own printable stl Files.

3D - Sheep (in Blender vs picoCAD2)

# Print Settings
* Printer:
    Creality Ender V3 KE
* Rafts:
    No
* Supports:
    Yes
* Infill:
    16%
* Notes:
    Printed with PLA on a heated bed.
